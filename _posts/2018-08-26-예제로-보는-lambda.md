# lambda

## foreach

### plain

~~~java
for(Transaction transaction: transactions) {
  transaction.setConfirmYn(true);
  transactionRepository.save(o);
}
~~~

### lambda

~~~java
transactions.forEach(o -> {
    o.setConfirmYn(true);
    transactionRepository.save(o);
});
~~~
